# 考试粥助手 - 操作手册

## 一、项目简介

考试粥助手是一款基于 AI 的本地化学习与考试辅助工具，帮助你：
- 上传学习资料（支持 PDF、DOCX、TXT、图片等格式，自动文本提取或 OCR 识别）
- AI 智能生成复习纲要和学习笔记
- 自动生成练习题（选择题、填空题、判断题、简答题）
- 刷题器模式：导入已有题目文件或试卷图片，AI 自动识别题目结构
- 追踪错题并进行针对性训练
- 按课程隔离管理所有学习数据

**所有数据完全存储在本地浏览器（IndexedDB），API Key 存储在 localStorage，不依赖任何云端服务。**

## 二、环境准备（逐步操作）

在安装本项目之前，你需要先准备好运行环境。请按顺序完成以下步骤。

### 2.1 安装 Node.js

Node.js 是运行本项目所必需的 JavaScript 运行时。

**第一步：下载 Node.js**

打开浏览器，访问 Node.js 官网：
```
https://nodejs.org/zh-cn
```

选择 **LTS（长期支持版）**，点击下载按钮。会自动下载适合你系统的安装包（Windows 为 `.msi` 文件）。

**第二步：运行安装程序**

1. 双击下载好的 `.msi` 安装文件
2. 点击 **「Next」**
3. 勾选 **「I accept the terms in the License Agreement」**，点击 **「Next」**
4. 安装路径保持默认（`C:\Program Files\nodejs\`），点击 **「Next」**
5. 在功能选择页面保持默认，点击 **「Next」**
6. **重要：** 如果出现 「Automatically install the necessary tools」 选项，**勾选它**
7. 点击 **「Install」**，等待安装完成
8. 点击 **「Finish」**

**第三步：验证安装**

安装完成后，按 `Win + R`，输入 `cmd`，回车打开命令提示符。输入以下命令：

```
node --version
```

如果显示类似 `v20.11.0` 的版本号，说明 Node.js 安装成功。

再输入：

```
npm --version
```

如果显示类似 `10.2.4` 的版本号，说明 npm（包管理器）也安装成功。

> 如果提示「不是内部或外部命令」，请重启电脑后重试。

### 2.2 确认浏览器

本应用推荐使用以下浏览器：

| 浏览器 | 最低版本 | 推荐 |
|--------|----------|------|
| Google Chrome | 80+ | ★★★ 最推荐 |
| Microsoft Edge | 80+ | ★★ 推荐 |
| Firefox | 78+ | ★ 可用 |

> Windows 11 自带的 Edge 浏览器即可使用，无需额外安装。

### 2.3 准备 AI API 密钥

本应用需要调用 AI 接口才能使用智能功能（生成题目、纲要、笔记、OCR）。你需要至少准备一个 AI 服务的 API Key。

**推荐选择（任选其一）：**

| 服务商 | 注册地址 | 费用 | 说明 |
|--------|----------|------|------|
| DeepSeek | `https://platform.deepseek.com` | 按量付费，价格低 | 国内直连，无需翻墙 |
| 通义千问 | `https://dashscope.console.aliyun.com` | 有免费额度 | 阿里云旗下 |
| OpenAI | `https://platform.openai.com` | 按量付费 | 需要海外网络 |

以 DeepSeek 为例的注册流程：
1. 访问 `https://platform.deepseek.com`，注册账号
2. 登录后进入控制台
3. 点击左侧 **「API Keys」**
4. 点击 **「创建 API Key」**
5. 复制生成的密钥（以 `sk-` 开头），**妥善保存**
6. 在 **「充值」** 页面充入少量余额（如 10 元）

> 请妥善保管你的 API Key，不要分享给他人。API 调用会产生费用。

## 三、安装与启动（逐步操作）

### 3.1 打开终端

**方法一：** 在项目文件夹中，点击地址栏，输入 `cmd`，回车

**方法二：** 按 `Win + R`，输入 `cmd`，回车。然后切换到项目目录：
```
cd C:\Users\你的用户名\Desktop\meoo_zip_1779717855429
```

**方法三（推荐）：** 按 `Win + X`，选择 **「终端」** 或 **「Windows PowerShell」**，然后切换到项目目录：
```
cd C:\Users\你的用户名\Desktop\meoo_zip_1779717855429
```

> 将上面的路径替换成你实际的项目文件夹路径。

### 3.2 安装项目依赖

在终端中输入以下命令，然后按回车：

```
npm install
```

**等待过程说明：**
- 首次安装需要下载约 200MB 的依赖包，耗时 2-5 分钟（取决于网速）
- 你会看到不断滚动的下载进度
- 可能出现黄色的 `WARN` 警告，这是正常的，不影响使用
- 安装完成后，会显示 `added xxx packages` 的提示
- 项目文件夹中会多出一个 `node_modules` 文件夹

**如果安装失败：**
```
# 如果因网络问题失败，可以切换到国内镜像源后重试：
npm config set registry https://registry.npmmirror.com
npm install
```

### 3.3 启动开发服务器

安装完成后，在终端中输入：

```
npm run dev
```

**启动成功的标志：**
终端会显示类似以下内容：
```
<i> [webpack-dev-server] Project is running at:
<i> [webpack-dev-server] On Your Network (IPv4): http://xxx.xxx.xxx.xxx:3015/
<i> [webpack-dev-server] On Your Network (IPv6): http://[::]:3015/
...
webpack compiled successfully
```

看到 `compiled successfully` 就表示启动成功。

> **注意：** 启动后终端窗口不要关闭！关闭终端 = 关闭应用。

### 3.4 打开应用

打开浏览器（推荐 Chrome 或 Edge），在地址栏输入：

```
http://localhost:3015
```

按回车，你应该能看到考试粥助手的首页。

### 3.5 停止服务器

当你不再使用时，回到终端窗口，按 `Ctrl + C`，输入 `Y` 确认即可停止服务。

### 3.6 日常使用

以后每次使用只需要两步：

```
# 第一步：打开终端，进入项目文件夹
cd C:\Users\你的用户名\Desktop\meoo_zip_1779717855429

# 第二步：启动服务
npm run dev
```

然后打开浏览器访问 `http://localhost:3015` 即可。

**不需要** 每次都运行 `npm install`，只有首次或更新项目后才需要。

### 3.7 生产构建（可选）

如果你想将应用打包成静态文件以便部署到其他地方：

```
npm run build
```

构建产物会输出到项目根目录的 `dist/` 文件夹中，可以直接用任何静态文件服务器托管。

## 四、首次配置（必须）

### 4.1 配置 AI API

1. 打开应用后，点击导航栏 **「设置」**
2. 在 **「API 配置」** 区域点击 **「添加 API 配置」**
3. 填写以下信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| 配置名称 | 自定义名称 | `DeepSeek主账号` |
| API Key | 你的 API 密钥 | `sk-xxxxxxxxxx` |
| API Base URL | API 服务地址 | `https://api.deepseek.com` |
| 模型名称 | 使用的模型 | `deepseek-chat` |
| Temperature | 生成随机性（0-2） | `0.7` |
| Max Tokens | 最大生成长度 | `2048` |

4. 点击 **「保存」** 按钮
5. 在 **「功能 API 分配」** 中为各功能选择对应的 API 配置
6. 点击页面底部 **「保存设置」**

> **OCR 识别**需要多模态模型（如 `gpt-4o`、`deepseek-vision` 等），普通文本模型不支持图片识别。

### 4.2 支持的 API 服务

| 服务商 | Base URL | 推荐模型 |
|--------|----------|----------|
| DeepSeek | `https://api.deepseek.com` | `deepseek-chat` |
| OpenAI | `https://api.openai.com` | `gpt-4o` |
| 通义千问 | `https://dashscope.aliyuncs.com/compatible-mode` | `qwen-plus` |

## 五、功能使用指南

### 5.1 课程管理

**入口：** 导航栏 → 课程

1. 应用预置了 8 个默认课程（高等数学、大学物理、有机化学等）
2. 点击 **「加入课程」** 进入某个课程
3. 点击 **「设为当前」** 激活课程
4. 点击 **「创建课程」** 可自定义新课程（名称、学科、难度）

> 激活课程后，所有上传的文档、生成的题目、错题等都会关联到当前课程。

### 5.2 知识库管理

**入口：** 导航栏 → 知识库

**上传文档：**
1. 点击上传区域或拖拽文件
2. 支持多文件同时上传
3. 文件处理：
   - **PDF** → 自动提取文本（PDF.js）
   - **DOCX** → 自动提取文本（mammoth）
   - **TXT / MD / CSV / JSON / XML / HTML** → 直接读取文本
   - **图片（JPG/PNG等）** → 弹出 OCR 识别窗口

**OCR 图片识别：**
1. 上传图片后弹出 OCR 窗口
2. 可选择开启「自动纠错」
3. 点击 **「开始识别」**，等待 AI 返回结果
4. 确认后点击 **「保存到知识库」**

**搜索与删除：**
- 使用搜索栏按文件名筛选
- 点击文档卡片上的删除按钮移除文档

### 5.3 复习助手

**入口：** 导航栏 → 复习助手

分为两个标签页：

**复习纲要：**
1. 勾选一个或多个文档
2. 点击 **「生成复习纲要」**
3. AI 根据文档内容和学科类型生成结构化纲要
4. 支持 Markdown 渲染（表格、列表、数学公式）
5. 点击 **「导出」** 可下载为 `.md` 文件

**学习笔记：**
1. 操作同上，生成详细的学习笔记
2. 包含知识点提炼、重难点解析、记忆技巧等

### 5.4 智能答题

**入口：** 导航栏 → 智能答题

智能答题分为两种模式，通过顶部标签切换：

#### AI 生成模式

1. 选择一个文档作为出题素材
2. 配置：
   - 题目数量（1-50 题）
   - 难度等级（1-5）
   - 题型选择（选择题、填空题、判断题、简答题）
3. 点击 **「生成题目」**
4. 生成的题目会标注来源信息，如 `AI生成 · 文档名 · 难度3`

#### 刷题器模式

导入已有的题目文件，AI 自动识别题目结构并补全缺失的答案和解析。

1. 切换到 **「刷题器」** 标签
2. 选择题目来源：
   - **上传文件：** 支持 `.txt, .pdf, .docx, .md, .jpg, .png` 等格式
   - **选择已有文档：** 从知识库中选择
3. 点击 **「查看推荐格式」** 可查看题库文件的推荐格式
4. 点击 **「识别并导入题目」**
5. 图片文件和扫描版 PDF 会自动调用多模态 AI 进行 OCR 识别
6. 导入成功后显示题目总数和各题型数量

#### 答题流程

1. 在「开始练习」区域可以：
   - **来源筛选：** 按题目来源过滤（如只看某次导入的题目）
   - **只做未完成：** 勾选后只显示未答过的题目
   - 查看完成进度（已完成 X 道 / 共 X 道）
2. 点击 **「开始练习」** 进入答题界面
3. 左侧 **答题卡** 显示所有题目编号，可点击跳转：
   - 绿色 = 答对
   - 红色 = 答错
   - 灰色删除线 = 已从题库移除
   - 默认 = 未答
4. 答题方式：
   - **选择题：** 点击选项
   - **判断题：** 点击「正确」或「错误」按钮
   - **填空题/简答题：** 在输入框中填写（简答题由 AI 智能判卷）
5. 点击 **「提交答案」** 查看对错和解析
6. 提交后可以：
   - 点击 **「这题么麻达」** 将该题从题库中永久删除（已掌握无需保留）
   - 点击 **「下一题」** 继续答题
7. 错误的题目自动加入错题本
8. 练习结束后显示成绩统计

### 5.5 错题本

**入口：** 导航栏 → 错题本

**功能：**
- 查看所有错题及统计（总错题数、待掌握、已掌握、高频错题）
- **错题重练：** 重新练习所有未掌握的错题
- **针对性训练：** AI 根据薄弱知识点生成新的练习题，来源标注如 `错题专项训练 · 微积分、导数`
- **标记已掌握：** 掌握后点击「已掌握」从待做列表移除

### 5.6 个人中心

**入口：** 导航栏 → 个人中心

- **自定义头像：** 点击头像区域上传图片，自动裁剪为合适大小
- 设置昵称和邮箱
- 查看学习统计：学习时长、答题数、正确率、资料数
- 等级系统：根据答题和上传资料积累经验值
- 最近活动记录
- 页面底部显示作者信息

### 5.7 设置

**入口：** 导航栏 → 设置

- **API 配置：** 管理多个 AI API 账号
- **功能 API 分配：** 为不同功能指定不同的 API（如聊天用 DeepSeek，OCR 用 GPT-4o）
- **本地数据存储：** 查看 IndexedDB 存储信息
- **数据隔离：** 开关课程数据隔离功能
- 页面底部显示作者信息

## 六、数据存储说明

| 数据类型 | 存储位置 | 说明 |
|----------|----------|------|
| 文档、题目、错题、笔记等 | IndexedDB (`StudyAssistantDB`) | 浏览器本地数据库 |
| 上传的文件原始数据 | IndexedDB (Blob) | 与文档记录一起存储 |
| API Key、用户设置 | localStorage | 浏览器键值存储 |
| 用户昵称、邮箱、头像 | localStorage | 浏览器键值存储 |

**注意事项：**
- 清除浏览器数据（缓存/Cookie）会删除所有学习记录
- 不同浏览器之间数据不共享
- 建议定期导出重要的笔记和纲要（使用导出功能）

## 九、常见问题

**Q: AI 功能不工作？**
A: 请检查设置页面中 API Key、Base URL 和模型名称是否正确填写，并确保点击了「保存设置」。

**Q: PDF 文件无法提取文本？**
A: 扫描版 PDF（图片型）无法直接提取文本。在智能答题的刷题器模式中上传扫描 PDF，系统会自动渲染为图片并调用多模态 AI 识别；在知识库中请使用 OCR 功能。

**Q: 图片题目如何导入？**
A: 在智能答题页面切换到「刷题器」模式，直接上传试卷截图或照片（JPG/PNG），系统会自动调用多模态 AI 识别并提取题目。

**Q: OCR 识别失败？**
A: OCR 需要多模态模型。请确保 OCR 功能配置的 API 使用的是支持视觉的模型（如 `gpt-4o`）。

**Q: 数据丢失了？**
A: 数据存储在浏览器 IndexedDB 中。如果清除了浏览器数据、使用了无痕模式、或更换了浏览器，数据将不可恢复。

**Q: 如何在不同设备间同步数据？**
A: 当前版本为纯本地存储，不支持跨设备同步。重要内容建议使用导出功能备份为 Markdown 文件。

**Q: 能否使用国内 AI 服务？**
A: 支持任何兼容 OpenAI 格式的 API 服务。填写对应的 Base URL 和模型名即可。

**Q: 「这题么麻达」是什么意思？**
A: 答题后点击此按钮表示你已完全掌握这道题，将其从题库中永久删除，不再出现在后续练习中。
