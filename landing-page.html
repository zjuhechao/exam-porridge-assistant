知识库管理

上传 PDF、Word、TXT、Markdown 等格式的学习资料，AI 自动提取文本内容，集中管理所有课程文档。

复习纲要 & 学习笔记

AI 根据文档内容自动生成结构化的复习纲要和详细的学习笔记，涵盖知识点归纳、重点难点解析。

智能出题

支持选择题、填空题、判断题、简答题等多种题型，难度可调，AI 根据学习内容自动生成配套练习题。

错题本

自动收集答题中的错题，标记掌握程度，支持基于错题重新出题强化，针对性弥补知识薄弱点。

🔍

刷题器 & OCR

导入已有试卷或题库，AI 自动识别题目结构并补全答案和解析。支持图片 OCR 识别纸质资料。

数据全本地

快速上手

三步开始使用

1

配置 AI API

打开设置页面，填写你的 API Key（支持 DeepSeek / OpenAI / Claude 等兼容接口）。Base URL 填入你的 API 服务地址，如 https://api.deepseek.com，模型名填 deepseek-chat 等。

2

上传学习资料

在知识库中创建课程并上传学习资料（支持 PDF / Word / TXT / Markdown / 图片）。AI 会自动提取文本内容，作为后续生成题目和笔记的基础。

3

开始学习

使用复习助手生成纲要和笔记，在智能答题中选择文档和题型进行练习，答错的题目自动收入错题本，针对性地巩固薄弱环节。
